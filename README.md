# Loan Status Predictor System using Machine Learning

This project builds a **Machine Learning model** to predict whether a loan application should be **approved or rejected** based on applicant information such as income, credit history, loan amount, and more.

The project includes:
- Data loading & understanding
- Data cleaning and preprocessing
- Handling missing values
- Categorical encoding (Label Encoding / One-Hot Encoding)
- Feature scaling
- Train–test split
- ML model building (Logistic Regression / Random Forest / etc.)
- Model evaluation using accuracy, confusion matrix, precision, recall, F1-score

---

## Key Features
### Data Preprocessing  
- Null value treatment  
- Encoding categorical variables  
- Scaling numerical features  

### Model Training  
- Logistic Regression / Random Forest  
- Hyperparameter tuning (optional)

### Model Evaluation  
- Accuracy Score  
- Confusion Matrix  
- Classification Report

The model predicts whether a loan **will be approved** based on applicant details, helping build intuition for **classification workflows in Machine Learning**.
